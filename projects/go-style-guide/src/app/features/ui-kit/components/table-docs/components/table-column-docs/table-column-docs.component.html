Implementing Table Columns

GoTableColumnComponent is required to render columns within go-table and is exported within GoTableModule. go-table-column cannot render outside of go-table.

Bindings

field

Defines data to be shown by linking column to the appropriate key within object defined in tableData when constructing GoTableConfig. For example if tableData was set to: and we wanted "John" to show, then field would be set to name.first.

sortable

This binding defines if sorting is enabled for the column. For this binding to work, GoTableConfig must have sortable set to true. By default sorting is enabled when sortable is set to true within GoTableConfig.

sticky

This binding sets the column as sticky so that when a user scrolls the column will stay on the page. This does not work on IE, but will fail gracefully and the columns will just be scrollable. For this to work properly the sticky columns have to be the first columns in the table.

title

This binding sets the column title.

width

This binding sets the width for column. The value must be in pixels.
