Portal

Usage

A portal gives you the ability to render dynamic content into any part of the UI.


It does this with the help of two directives: goPortalAttachTo and goPortalTarget.


The goPortalAttachTo is the template that you want to render and the goPortalTarget is the destination where you want to render the template into.

Setup

To use the portal feature anywhere in the app, you must follow the following steps:

  1. Specify the template for the goPortalAttachTo directive.

  2. Then, specify the destination via the goTarget directive.
